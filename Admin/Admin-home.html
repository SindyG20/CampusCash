Steps to Manage Students

1
Access Student Records

Navigate to the Students section from the main dashboard.

2
Search or Filter

Use the search bar or filters to find specific students.

3
View/Edit Details

Click on any student to view full details and edit information.

Important Notes

  • Always verify student ID before making changes to records
  • Maintain confidentiality of student information
  • Document any changes made to student records
  • Contact IT support if you encounter system issues
